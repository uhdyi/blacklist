This is a fun project mainly for experiencing web service. The API involoved in this application are: Amazon Product Advertising API, Google(Youtube) API, Flickr API. The next steps would be Twitter API, Facebook API and Yahoo Finance API.

To use this application, select the search category form the drop down list, then input keywords in the textbox, click 'Go' button, it will show searched results from amazon(books and music), youtube(video) or flickr(photos). There is some problem for Justhost.com, which is the place this application is hosted, to connect to google, the Video seach cannot work.
